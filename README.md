# Ksh

Ksh (Konrad's Shell, pronounced 'keysh') is an interactive shell, similar to Bash. It was built by following the
["Build Your Own Shell" Challenge on CodeCrafters](https://app.codecrafters.io/courses/shell/overview).

> In this challenge, you'll build your own POSIX compliant shell that's capable of
> interpreting shell commands, running external programs and builtin commands like
> cd, pwd, echo and more. Along the way, you'll learn about shell command parsing,
> REPLs, builtin commands, and more.
